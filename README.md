# linux-sysadmin-lab

Reproducible labs and small projects for Linux SysAdmin work using Ubuntu Server and Docker.

## Repository structure
- `docs/`: documentation and lab notes
- `projects/`: reproducible docker-based projects

## How projects are used
Each project includes:
- a `docker-compose.yml`
- optional `.env` configuration
- scripts to start and stop services

Projects are designed to be executed on Ubuntu Server.
